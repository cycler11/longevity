# Изображение для Meditation Event

- **Название:** Meditation & Mindfulness Event
- **Luma ссылка:** https://luma.com/iz5nfed1
- **Дата:** TBA (будет обновлена позже)

## Как добавить изображение с Luma:

1. Откройте страницу события: https://luma.com/iz5nfed1
2. Найдите изображение события на странице
3. Скачайте изображение
4. Сохраните его как `meditation-event.jpg` в папку `/public/events/`
5. Обновите файл `src/data/events.ts`:
   - Найдите строку с `photo: "/events/default.png"`
   - Замените на `photo: "/events/meditation-event.jpg"`

Или просто переименуйте скачанное изображение в `meditation-event.jpg` и поместите в `/public/events/`, затем обновите путь в коде.
