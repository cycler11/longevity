# Как запустить сайт

## 1. Разработка (рекомендуется)

В терминале в папке проекта:

```bash
cd /Users/cyclerss/Downloads/CLC-0502
npm run dev
```

Дождись сообщения **Ready** в консоли, затем открой в браузере:

**http://127.0.0.1:3000**

(Именно этот адрес, не localhost и не https.)

## 2. Продакшен

```bash
cd /Users/cyclerss/Downloads/CLC-0502
npm run build
npm run start
```

Или одной командой: `npm run serve`

Открой: **http://127.0.0.1:3000**

## Ссылки

- Главная: http://127.0.0.1:3000
- Контакты: http://127.0.0.1:3000/contacts
- Медиа: http://127.0.0.1:3000/media
- События: http://127.0.0.1:3000/events

## Если не открывается

1. Убедись, что в консоли есть строка `✓ Ready` и нет красных ошибок.
2. Открывай именно **http://127.0.0.1:3000** (не https, не localhost).
3. Если порт 3000 занят, запусти: `npm run dev:3001` и открой http://127.0.0.1:3001
